# sql_database_task_1
# PostgreSQL Table Practice

## Overview
This repository contains my **SQL practice task using PostgreSQL**.  
In this task I created a database and designed multiple tables using different SQL data types.  
Each table also contains several **INSERT queries** to add sample data.



## Database Information
**Database Name:** `company_db`  
**Database System:** PostgreSQL  
**Total Tables:** 10  
**Rows per Table:** 50  
**Total Rows Inserted:** 500  



## Tables Included
The database contains the following tables:

- Employees  
- Departments  
- Products  
- Customers  
- Orders  
- Suppliers  
- Payments  
- Inventory  
- Attendance  
- Reviews  



## Table Structure
Each table has **6 columns** and uses different SQL data types such as:

- `INT`
- `VARCHAR`
- `DATE`
- `DECIMAL`
- `BOOLEAN`
- `TIME`
- `TEXT`



## Data Insertion
Each table contains **50 INSERT queries** to populate the data.

Example:

```sql
INSERT INTO employees VALUES (1,'John',28,'IT',45000,'2024-01-01');
```
